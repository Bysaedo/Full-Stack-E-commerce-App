Full-Stack E-Commerce Application

A full-stack e-commerce platform built with ASP.NET Core, Entity Framework, SQLite, React, and Stripe.
Includes real payment processing, product management, and a complete checkout workflow.

Features

Backend (ASP.NET Core + EF Core):

-RESTful API architecture
-SQLite database using Entity Framework Core
-One-to-many data relationships
-Product catalog and product search
-Order creation and management
-Secure API key handling
-End-to-end Stripe payment integration

Frontend (React):

-React application with routing
-Shopping cart with state management
-Product browsing and detail views
-Checkout flow integrated with Stripe
-Order confirmation and UI responses

Tech Stack

Frontend: React, React Router
Backend: ASP.NET Core, Entity Framework Core
Database: SQLite
Payments: Stripe API
Other: Axios, CORS configuration

Setup

Backend: 
-dotnet restore
-dotnet run --project API

Frontend: 
-cd client
-npm install
-npm run dev

Summary

This project demonstrates the development of a full-stack e-commerce solutionwith real payment processing,
secure API design, database modeling, and a professional checkout experience. 
It follows modern practices for both frontend and backend development.
